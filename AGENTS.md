# Lalawash — Laundry Service Landing Page

## Stack
- Pure HTML5 + CSS3 + Vanilla JS
- No frameworks, no build tools, no npm
- Mobile-first responsive design

## Directory
```
~/Code/lalawash/
├── index.html          # Main landing page
├── css/
│   └── style.css       # All styles
├── js/
│   └── main.js         # Vanilla JS (optional)
└── assets/
    └── images/         # Images
```

## Design direction
- Filipino laundry brand — modern, clean, trustworthy
- Warm/earthy tones (di puro puti)
- Mobile-first
- Fast loading (< 3s on 3G)
- No external dependencies except Google Fonts (optional)

## Commands
- Open `index.html` in browser to preview
- No build step needed
